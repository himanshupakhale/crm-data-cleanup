CRM Data Cleanup & Deduplication

Overview
This project demonstrates cleaning and standardizing CRM customer data using Python and Pandas.

Tools
- Python
- Pandas
- GitHub

Dataset
Sample CRM dataset in CSV format.

Cleaning Steps
- Removed duplicate records
- Removed missing values
- Validated email format using regex
- Standardized phone numbers

Output
Generated cleaned dataset: customers_cleaned.csv
